Setup Required

To access the admin portal, Netlify Identity needs to be enabled first. Please follow these steps:

  1. Go to your Netlify site settings ‚Üí Identity
  2. Click "Enable Identity" if it's not already enabled
  3. Under "Registration preferences", select "Invite only"
  4. Under "Git Gateway", click "Enable Git Gateway"
  5. Optional: Go to "Identity ‚Üí Users" and invite yourself with your email
  6. Return to this page and refresh
